1. Primitive Data Types:
------------------------
1; String: Represents a sequence of characters.

2: Number: Represents both integer and floating-point numbers.

3: BigInt: Represents integers with arbitrary precision.

4:Boolean: Represents a logical entity and can have two values: true or false.

5: Undefined: Represents a variable that has been declared but not assigned a value.

6: Null: Represents the intentional absence of any object value. It is one of the primitive types.

7: Symbol: Represents a unique identifier; often used for object properties to avoid name clashes.

-----------------------------------------
